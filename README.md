# open_project_directory.aux2

[English](README.en.md) | [日本語](README.md)

[![AviUtl2 Catalog](https://aviutl2-catalog-badge.sevenc7c.workers.dev/badge/v/sevenc-nanashi.open_project_directory-aux2)](https://aviutl2-catalog-badge.sevenc7c.workers.dev/package/sevenc-nanashi.open_project_directory-aux2)

AviUtl2の編集メニューに、現在のプロジェクトファイルが存在するフォルダをエクスプローラーで開く項目を追加するAviUtl2の汎用プラグインです。

## インストール

[Releases](https://github.com/sevenc-nanashi/open_project_directory.aux2/releases/latest)から`open_project_directory-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
