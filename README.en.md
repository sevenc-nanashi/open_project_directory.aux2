# open_project_directory.aux2

[English](README.en.md) | [日本語](README.md)

[![AviUtl2 Catalog](https://aviutl2-catalog-badge.sevenc7c.workers.dev/badge/v/sevenc-nanashi.open_project_directory-aux2)](https://aviutl2-catalog-badge.sevenc7c.workers.dev/package/sevenc-nanashi.open_project_directory-aux2)

A generic plugin for AviUtl2 that adds an Edit menu item to open the folder containing the current project file in Explorer.

## Installation

Download `open_project_directory-v<version>.au2pkg.zip` from [Releases](https://github.com/sevenc-nanashi/open_project_directory.aux2/releases/latest), and drag-and-drop it into the AviUtl2 preview.
